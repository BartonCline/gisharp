The following can be coppied to Run > Custom Command > Execute in the project
settings in order to debug CodeGen:

GLib-2.0:
${TargetFile} --repository=GLib-2.0 --fixup=${SolutionDir}/GLib-2.0/GLib-2.0.girfixup --output=${SolutionDir}/GLib-2.0/Generated.cs --assemblies='${SolutionDir}/Core/bin/${ProjectConfig}/Core.dll:${SolutionDir}/Core/bin/${ProjectConfig}/NativeLong.dll'

GObject-2.0:
${TargetFile} --repository=GObject-2.0 --fixup=${SolutionDir}/GObject-2.0/GObject-2.0.girfixup --output=${SolutionDir}/GObject-2.0/Generated.cs --assemblies='${SolutionDir}/Core/bin/${ProjectConfig}/Core.dll:${SolutionDir}/Core/bin/${ProjectConfig}/NativeLong.dll:${SolutionDir}/GLib-2.0/bin/${ProjectConfig}/GLib-2.0.dll'
